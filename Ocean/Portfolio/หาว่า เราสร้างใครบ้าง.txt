Objectives
เพื่อหาจำนวน ตัวที่ที่สร้างใหม่ตาม รหัสตัวแทน 7 หลัก และ งวด

Process Overview

Preconditions

Process Description

input
agent_code    : รหัสตัวแทน 7 หลัก
select_period : งวดที่ทำการค้นหา

output
agent_count : จำนวน ตัวแทนที่สร้างใหม่

Detail

select  count(distinct ad.sales_code) as agent_count
from sales_team_structure ma inner join sales_team_structure  ad on ma.id = ad.recruiter 
where ma.sales_code = :agent_code and ma."period" = :select_period


return agent_count
