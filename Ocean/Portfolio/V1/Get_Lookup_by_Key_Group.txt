Objectives
ดึงข้อมูลเกี่ยวกับ key และ value ตาม key_group และ period ที่ได้รับมา (และส่งชื่อ field ที่ต้องการเรียง คือ key หรือ value)
 

Process Overview
ดึงข้อมูลเกี่ยวกับ key และ value ในฐานข้อมูล Lookup ตาม key_group, period และส่งชื่อ field ที่ต้องการเรียง คือ key หรือ value ที่ได้รับมา
 

Preconditions
รับค่า key_group, period เข้ามา
 

Process Description
ให้ get key, value จาก Query นี้
Icon
select		key_group, key, value
from		agpt_lookup_data 
where		key_group = v_key_group -- 'ov'
	and	period_from <= v_period_number and period_to >= v_period_number -- '256018'
order by	v_order_by; -- key
     

Post-conditions
ทำการ return ข้อมูลทั้งหมด 3 fields คือ 
- key_group
- key
- value
